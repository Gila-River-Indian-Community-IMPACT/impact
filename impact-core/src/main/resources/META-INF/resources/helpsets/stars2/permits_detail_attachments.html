Attachments

Overview

The Attachments third-level menu of the Permit Detail page allows you to view, add, delete and edit attachments associated with the permit process. It is IMPORTANT to note that these attachmnents are in support of the permit and ARE NOT included in the issued permit document. The types of attachments are not limited and may include documents that support data, data delivery, calculations, EACs, process flows or other documents deemed necessary to complete the permit process. Keep in mind that all attachments added to the Permit Detail are a public record; therefore, trade secret or sensitive documents should not be included in this section of the system.

The header at the top of the Attachments page is the same header viewed on the Permit Detail page and identifies specific information regarding the permit; most importantly the Permit Number, Permit Type and issuance dates for various drafts, public notices, comment periods, effective dates and expiration dates, when applicable. Under the header is a datagrid that lists all Attachments associated with the permit action.

Attachments Datagrid

Under the header is a datagrid with the following information:

  • Attachment ID: a unique ID number assigned to the attachment. The Document ID is a hyperlink for editing the Attachment Type or Description or deleting the attachment.

  • Description: text describing the attachment. This is entered at the time the document is uploaded. The Description is a hyperlink for viewing the attachment.

  • Modified By: the person who uploaded the attachment.

  • Last Modified: the date the attachment was uploaded.

Uploading an Attachment

To add an attachment click the Add button at the bottom of the page which will generate the following pop-up:

  • Attachment Type: select the type of attachment from the pick list.

  • Description: enter an appropriate Description of the attachment.

  • File to upload: Choose the file you would like to attach to the submittal from your computer.

Click  to populate the information into the datagrid. Clicking  will cancel any attachment information you just entered electronically.

The datagrid will be automatically populated with an Attachment ID (unique ID number automatically generated by the system and hyperlink for editing the Attachment Type or Description or deleting the attachment), Description (hyperlink to access attachment), Modified By and Last Modified (date the attachment was last uploaded).

Create a New Facility

Overview

The system's Create Facility page allows you to enter the new facility's address so that the system can assign a Facility ID to it and create an initial Facility Profile. The system then gives you access to pages in which you may enter additional information about the facility. Entering data in the fields on these pages is the first step in the process of building the Facility Profile.

IMPORTANT: Before creating a facility in the system, you must first ensure that the facility doesn't already exist. You can accomplish this via a two step process. First, click on the Facility Tab and search for the facility by name. The Help on that page will provide you additional guidance if needed. If the facility is not found in the system, you must then search for the facility in the Core database. This is accomplished by giving the facility information to a Central Office Facility Reconciler and requesting they search for the facility in Core. If the facility is not found, proceed as described below. If the facility is found, you must record the Core Place ID, after you have successfully created a facility as described below, proceed following instructions and enter the Core Place ID into the Facility Information page. FAILURE TO FIND AND ENTER AN EXISTING CORE PLACE ID IN THE SYSTEM WILL CAUSE YOU TO LOSE ALL DATA ENTERED FOR THIS FACILITY, INCLUDING BUT NOT LIMITED TO APPLICATIONS, EMISSIONS REPORTS, ETC.

Asterisks(*) denote input fields that the system requires for creating a facility. However, information in some other input fields that may not be required, may be needed for the system to perform other functions. Therefore, completing as many fields as possible with correct information benefits the system and all its users.

Establishing a New Facility ID

Once the Create Facility page opens, several information fields will appear. Enter the facility data into all input fields. The Facility Name should be entered as it is to appear in the system and on all documentation. The facility’s physical Address should be entered; other addresses may be entered in the system's Owner/Contact page once this Create Facility page is completed. If the source is portable select the check box.

If you choose to cancel/delete the information entered, click the Cancel/Reset button to delete all the information on the page.

Once all the required input fields have been completed, click on the Submit Create Facility button. The page will then populate the information into a Facility Profile, automatically assign a Facility ID and refresh to the Facility Detail page. This information you have entered will appear in the information section of the Facility Profile page. If the facility exists in Core, remember to enter the Core Place ID into the Facility Information page.

Once the Facility Detail page appears, you must continue building a new facility inventory. If you continue to need assistance, click the Help button on the Facility Detail page.

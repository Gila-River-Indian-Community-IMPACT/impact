Entering and Editing Data

When you enter or edit data, you will work with input fields of various types and some standard edit buttons.  The pages in this section cover these topics, including all the sub-topics listed under them below, as well as Exception Reporting, which is how the system lets you know that the data you have entered is either incomplete, inconsistent or both. There are several types of validation that may result in exceptions being reported to you. The pages below will cover all of them.

This information will make it easier for you to enter data, whether for permit applications, emissions inventories or compliance reports, or to keep the system's view of your facility up-to-date. This section, then, covers the following topics:

  • Input Fields
    • Text
    • Pick lists
    • Dates
    • Check boxes
    • Radio buttons
    • Entries in lists
    • Data shuttles
  • Edit Buttons Operations
    • Edit
    • Save, Cancel
    • Add, Delete
    • Apply, Create
    • Validate
  • Exception Reporting, in the case of
    • Simple field validation exceptions
    • Required field exceptions
    • Field dependency exceptions
    • Complex form validation exceptions

The help topic pages listed above are available via the Help Table of Contents, which is accessible by clicking on the tab or the button on the individual help topic pages.

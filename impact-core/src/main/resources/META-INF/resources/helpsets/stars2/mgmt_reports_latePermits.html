Late Permits

The system’s second-level menu, Late Permits, of the Mgmt Reports tab allows you to search permits in the system with a Warning or Danger status for the associated permitting workflow. The Warning or Danger status is assigned to permits based on a combination of several factors: DO/LAA(s) to which the permit belongs, Types of permit(s) (including historical types such as registration), and the Reason for the permit.

Late Permits Search Criteria

You may search for Late Permits on the basis of any or all of the criteria listed below. You can fill in or choose a value for whichever fields you want to use to restrict your search.

  • DO/LAAS: Select from a picklist of District Offices and Local Area Agencies, for example, Ohio EPA AQD, Southwest District Office or Canton City Health Department.
  • Types: Select from a picklist of permit types, for example, PBR or Title V Permit.
  • Reason: Select from a picklist of permit reasons, for example, Renewal or MPM.
  • General Permit: Select this radio button to filter for only general permits that meet the criteria you’ve selected above.
  • Express Status: Select this radio button to filter for only permits being process under the express processing status that meet the criteria you’ve selected above.
  • Display Status: By default, the report searches only permits that have Permit Status of Danger. Select this radio button to filter for all in process permits.

If you select various fields and decide you want to change your selections and start over, you can click  to clear the selections. Note, the picklists above have the ability to select multiple items in the list by holding your “Ctrl” key while clicking on the list items.

By default, the search criteria are specified to select all Late Permits. You may of course modify these values before you submit your search. The more criteria you are able to specify, the more precise your search will be. Broad searches can retrieve many records that can make it difficult to home in on the information you are really looking for.

Late Permits Search Results

After you complete the criteria for the search you want, and click on , the system searches all of the records for all the permits stored in its database, selects the ones that meet ALL of the criteria you have specified, and returns summary information about the Late Permits in a datagrid. The datagrid below is the result of a search submitted with search criteria specified to select all Renewal (Reason) PTIO permits with a Danger status for Akron Regional Air Quality Management District.

Late Permits Search Results Datagrid

The summary data returned includes columns providing Facility ID, Facility Name, Permit Number, Workflow Id, Total Days, Referred Days, Current Task, Received Date, Permit Type, General, Issue Draft, Express, Reason, Rush and Permit Status for all the Late Permits that matches your search criteria. The following summary data is further explained:

  • Facility Id: The unique ID number assigned to the facility. The Facility Id is a hyperlink for accessing the associated Facility Profile.
  • Permit Number: The unique ID number assigned to the permit document. The Permit Number is a hyperlink for accessing the associated Permit Detail page.
  • Workflow Id: The unique ID number assigned to the permitting workflow. The Workflow Id is a hyperlink for accessing the associated Workflow Diagram.
  • Total Days: The total number of days the permit has been in processing based on the Received Date and including the Referred Days.
  • Referred Days: The total number of days the permit has been referred to the company and is therefore not counted on the Ohio EPA ‘clock’.
  • Current Task: The Task Name that represents the current stage the permit is in the workflow.
  • General: Identifies either Y (yes) or N (no) as to whether the permit is a general permit.
  • Issue Draft: Identifies either Y (yes) or N (no) as to whether the permit has been issued in draft form.
  • Express: Identifies either Y (yes) or N (no) as to whether the permit is being processed as an express permit.
  • Rush: Identifies either Y (yes) or N (no) as to whether the permit is under rush status (for internal purposes).
  • Permit Status: Displays the status of the permit(s) with respect to lateness as Warning or Danger. The status is assigned to permits based on a combination of several factors: DO/LAA(s) to which the permit belongs, Types of permit(s) (including historical types such as registration), and the Reason for the permit.

As with all datagrids, information from the list may be viewed, exported to Excel or printed. By clicking  you may view and print. By clicking  a pop-up will appear in which you must click Open in order to view the information in Excel.

This datagrid supports all the common functionality for datagrids as described in Common User Interface Elements - Datagrids

Show Charts

Notice at the bottom of the datagrid a  button. Click this button to generate pie charts, above the datagrid, displaying data associated with datagrid. Three charts will be displayed:

  • Summary by DO/LAA: Identifies the number of late permits by DO/LAA. It will only display the DO/LAA’s which were selected as part of the search criteria.
  • Summary by Permit Type: Identifies the number of late permits by Permit Type. It will only display the permit types which were selected as part of the search criteria.
  • Summary by Permit Status: Identifies the number of late permits by Permit Status. It will only display the status for those permits identified in the datagrid.

Control Equipment Summary Information

Overview 

The system’s third-level menu, Control Equipment, of the Facility Profile page allows you to view a list of all control equipment located at a facility. This page also allows you to access a specific Control Equipment Information page, through a hyperlink for each piece of control equipment, where you can edit information pertaining to that control equipment. For more information on editing Control Equipment Information, refer to the Facility Tree Help page.

The following topics are contained in this help page:

  • Control Equipment List
  • Printable View and Exporting to Excel
  • Related Help Topics

Control Equipment List

The following datagrid of information will appear when you access the third-level menu, Control Equipment:

  • Company ID - The term the facility uses to identify the equipment. Clicking on the Company ID hyperlink will take you to the specific Control Equipment Information page for that control equipment.

  • Company Description – The term the facility uses to describe the equipment.

  • AQD Description - The description that is assigned by Ohio EPA and will be included in permits and on documentation within the system.

  • Equipment Type - The type of control as identified to Ohio EPA; for example; Flare or Web Scrubber.

  • Operating Status - Identifies the operating status as it exists with Ohio EPA; Operating or Not Operating.

  • Manufacturer - Identifies the manufacturer of the control equipment as it exists with Ohio EPA.

  • Model - Identifies the model from manufacturer of the control equipment as it exists with Ohio EPA.

  • Initial Installation Date - Identifies the initial install date when the control equipment was installed for the very first time, as it exists with Ohio EPA.

Information from the Control Equipment list may be viewed, exported to Excel or printed. By clicking  you may view and print. By clicking  a pop-up will appear in which you must click Open in order to view the information in Excel.

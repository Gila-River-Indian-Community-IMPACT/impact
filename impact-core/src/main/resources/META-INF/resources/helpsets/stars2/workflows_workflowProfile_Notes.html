Workflow Notes

Overview 

The Notes page allows you to add additional information to the current workflow that may be relevant to the process.  These Notes are viewable only from the internal side of the system; however, they are considered a public record.



    Notes List

    The Notes section may have information in the datagrid produced by other internal users or by the system itself. The datagrid contains the following information:

    • Note ID- The identification number assigned to the note by the system. A hyperlink for viewing the full note.
    • Note – The beginning portion of the note’s text.
    • User Name – The name of the person who created the note.
    • Date – The date that the note was created or last edited.

    Back to Top



    Entering a Note

    If you wish to document information in the workflow select Add Note at the bottom of the datagrid. A pop-up screen will appear. The pop-up contains a text box where you may enter your information. The beginning portion of the text will become the title of the Note in the datagrid.

    Once you have entered your information, select the Save button. If you decide not to enter the information then select the Cancel button at any time. Selecting either button returns you to the workflow Notes page.

    Back to Top



    Viewing a Note

    When you select a pickable Note ID in the Notes list, the system takes you to a pop-up window that displays the entire text of the note. The system will allow you to view any Note in the data grid; regardless of who entered the Note.

    Back to Top



    Editing a Note

    To edit or add text to a Note you created, select the pickable Note ID in the Notes List, make your changes and then select Save.

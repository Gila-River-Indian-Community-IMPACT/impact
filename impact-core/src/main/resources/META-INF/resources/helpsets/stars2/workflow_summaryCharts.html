Summary Charts

Overview 

This Help page contains two pie charts summarizing workflow information applicable to ALL workflows in the system. The first, titled “Open Process by Status”, displays the number of active workflows currently in each possible status: OK (on time/green); Jeopardy (nearing the deadline/yellow); or Late (past the deadline/red). The second, titled “Open Process by Type”, displays the number of active workflows of each possible workflow (by workflow name); for example Permitting, Blue Card Review, or Undelivered Mail.

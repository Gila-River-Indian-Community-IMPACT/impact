General Facility Attachments

Overview

The system’s third-level menu, Attachments, of the Facility Profile page allows you to view a list of attachments, edit or delete an attachment, or add a new attachment to the facility’s profile. The Attachments list is not a list of ALL attachments in the system that were incorporated for various purposes, such as applications or compliance reporting. Rather this third-level menu is a place where Ohio EPA staff can incorporate attachments that are relevant to the facility. For example, results from modeling, process flow diagrams or other important documents.

  • Attachments List
  • Adding an Attachment
  • Editing or Deleting an Attachment
  • Printable View and Exporting to Excel


Attachments List

The following datagrid of information will appear when you access the third-level menu, Attachments:

  • Attachment ID – A unique ID assigned by the system when the attachment is uploaded. Clicking on the Attachment ID hyperlink generate a pop-up where you can edit or delete the attachment. See the heading below for more information on “Editing or Deleting an Attachment”.

  • Attachment Type – Indicates the type of attachment as selected from a picklist when the attachment was uploaded; Air Toxics Modeling Results, Other, or Process Flow Diagram(s). Clicking on the Attachment Type hyperlink opens the attachment for viewing or downloading.
  • Description - The description entered into the system when the attachment is uploaded. 

  • Modified By – Indicates the person who uploaded, or last modified, the attachment.

  • Last Modified – Indicates the date when the person uploaded, or last modified, the attachment.

Add an Attachment

To add a new attachment, click  at the bottom of the datagrid. A pop-up will appear in which you must choose the Attachment Type (Air Toxics Modeling Results, Other, Process Flow Diagram), enter a Description of the attachment, and upload the file using the Browse button. Click  to add the attachment or  to abort the process and return to the Attachments page without uploading an attachment.

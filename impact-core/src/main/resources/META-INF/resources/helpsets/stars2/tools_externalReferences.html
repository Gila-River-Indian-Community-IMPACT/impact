External References

Overview

The system’s second-level menu, External References, of the Tools tab allows you to view a list of External References that a facility, or Ohio EPA staff person, may find helpful for completing tasks in Air Services. Headings divide the references into appropriate sections, such as Guidance, Tools or Rules. Each item is a hyperlink that will open a second browser window sending you to the internet location where the information is located, or to a document itself. For example, clicking PBR Supplemental Forms will send you to Ohio EPA’s permit-by-rule website where you can access Supplemental Forms that must be completed in order for you to submit a permit-by-rule notification through the system.

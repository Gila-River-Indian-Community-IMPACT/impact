Location

Overview

The system’s third-level menu, Facility Address History, of the Facility Profile page allows you to view a list of the current and historical addresses for the facility. The facility will have an address history if the facility has moved from one location to another, and the address in the Facility Detail was subsequently changed. The facility’s address can not be changed from this page, but rather through the Facility Profile. Refer to the Facility Detail Help page for information on changing an address.

Location List

The following datagrid of information will appear when you access the third-level menu, Facility Address History.

The facility’s historical addresses, if applicable, are listed by most recent. The current facility address should be listed first in the table. The information in the table includes, the address and the Start Date and the End Date, based on when the address changed.

Applicable Federal Rules

Overview

This tool is a quick reference item to allow you to view the following Federal Rules that have been identified as applicable for a facility thus far: MACT, NSPS, NESHAPs, PSD, NSR, 112(r), and Title IV. If the box is checked then the facility has been determined to be subject to that category. Subheadings for each of the Federal Rules will be displayed under the Federal Rules Applicability section that identifies applicable subparts and/or pollutants for each applicable Federal Rule.

Identifying Federal Rules as Applicable

Owners or operators applying for a PTI or PTIO air permit must designate in the application, whether submitted electronically or in hard copy form, if certain Federal Rules apply. If a Federal Rule has been identified as applicable in an application submitted electronically by the applicant, or entered electronically by DO/LAA staff in an application, it will be automatically incorporated in the Federal Rules third-level menu of the Facility Tab.

Ohio EPA staff have the ability to add to the Federal Rules Applicability section by selecting the Edit Rules button. The applicability can be added by selecting the check box next to the Federal Rule in question which will add a check mark. When a Federal Rule is added, a subheading for that Federal Rule will be generated under the Federal Rules Applicability section. Under this subheading you will identify for each Federal Rule applicable subparts and/or pollutants via a drop down menu. Click Add Subpart or Add Pollutant, as applicable, and select the subpart or pollutant from the drop down menu. Then select the Save button to save the data and be taken out of edit mode. Please note: external users do not have the capability of adding Federal Rules Applicability except through the application process.

Editing the Applicability of Federal Rules, Subparts or Pollutants

Ohio EPA staff have the ability to update the Federal Rules Applicability section by selecting the Edit Rules button. 

  • The applicability of the entire federal rule (and any associated subparts or pollutants) can be removed by de-selecting the check box next to the Federal Rule in question which will remove the check mark.
  • To change a selected subpart and/or pollutant, simply select a new one from the drop down list.
  • To remove a selected subpart and/or pollutant, place a check mark in the box next to the subpart and/or pollutant you wish to remove and click Delete Selected Subparts (or Pollutant).

Then select the Save button to save the data and be taken out of edit mode

Please note: external users do not have the capability of removing Federal Rules Applicability except through the application process.

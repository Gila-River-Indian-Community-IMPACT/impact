Issuance Report

The system’s second-level menu, Issuance Report, of the Mgmt Reports tab



allows you to view a summary of the issued permits for the various permitting options, by office for the specified time period.  Enter a From Date and To Date and click Submit.

The following Issuance Report datagrid will be generated with summary information regarding permit issuance, by DO/LAA, for the time period searched.



The resulting data set allows you to drill down to a specific set of permit records to view issuance details in the report.  By clicking on any of the hyperlinked issuance numbers in the datagrid for a given DO/LAA, a summary of the permit type/stages, by task assigned for each user in the selected DO/LAA, will be generated.

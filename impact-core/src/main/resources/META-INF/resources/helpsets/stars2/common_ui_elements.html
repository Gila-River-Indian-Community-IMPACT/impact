Common User Interface Elements

The screens in the system that give you access to the information and operations you need are designed to be consistent throughout, providing you with a common user interface for doing your work. Once you are familiar with the common elements of the system you can work easily in the entire system. This chapter describes many common aspects of the interface, including examples. It covers the following topics:

  • Datagrids
  • Detail Data

    • Edit Button Operations

      • Add, Delete
      • Create
      • Generate, Upload
      • Validate
    • Exception Reporting, in the case of ...
      • Simple Validation
      • Required fields

      • Complex Form Validation

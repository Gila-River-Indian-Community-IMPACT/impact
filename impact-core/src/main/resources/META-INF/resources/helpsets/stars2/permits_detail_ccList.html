Carbon Copy

Overview

The Carbon Copy third-level menu of the Permit Detail page allows you to send the permit to another party (e.g., a person interested in the permit issuance) once it has been issued from Central Office as a Draft or Final permit.  You only need to add another party to this list other than the “standard” carbon copy recipients (e.g., DO/LAA, USEPA, etc.).  Standard carbon copy recipients are automatically built into the system.

The header at the top of the Carbon Copy page is the same header viewed on the Permit Detail page and identifies specific information regarding the permit; most importantly the Permit Number, Permit Type and issuance dates for various drafts, public notices, comment periods, effective dates and expiration dates, when applicable.

Under the header is a datagrid that identifies all persons, with addresses, that receive a Carbon Copy action associated with the permit. You can edit, add or delete entries by clicking the Edit button.

  • To edit an entry simply type the new information into the appropriate text box.

  • To delete an entry select the box next to the name by placing a check mark in it. Click Delete Selected.

  • To add an entry click Add. Another row will be added where you can type the appropriate information into each of the text boxes.

After adding or deleting an entry, click Save Changes or Discard Changes. Selecting Discard Changes will cancel any information you just entered.

PBR Counts

The system’s second-level menu, PBR Counts, of the Mgmt Reports tab allows you to generate a report of the processing of permit-by-rule (PBR) notifications received by Ohio EPA. This topic contains the following sections:

  • PBR Counts Search Criteria
  • PBR Counts Search Results
  • Related Help Topics

PBR Count Search Criteria

You may search for PBR Counts between any date range by entering a From Date and To Date under the PBR Count Report Filter heading and clicking.  If you decide you want to change your selections and start over, you can clickto clear the dates.

PBR Counts Search Results

After you enter the dates and click on, the system searches all of the records for PBR notifications stored in its database, selects the ones in the date range you have specified, and returns summary information about the PBR Counts  in a datagrid.

The summary data returned is organized by DO/LAA with the following information for each office and each specific type of PBR available:

  • Total Count (ex. Superseded): This represents the total number of ALL types of PBR notifications acted upon (i.e., issued" where "issued" means accepted or denied) by that office for the date range specified, except any PBR notification that are superseded.
  • Received: The total number of PBR notifications received by that office for the date range specified for the specific PBR Type identified.
  • Accepted: The total number of PBR notifications with a Disposition of Accepted in the system by that office for the date range specified for the specific PBR Type identified. PBR notifications are reviewed on their respective Application Detail page where their Disposition is set.
  • Denied: The total number of PBR notifications with a Disposition of Denied in the system by that office for the date range specified for the specific PBR Type identified. PBR notifications are reviewed on their respective Application Detail page where their Disposition is set.
  • Superseded: Count of all PBRs that were superseded during the date range specified (i.e. another permit was issued for the emissions unit so that the PBR is no longer valid).
  • Average Processing Time (Days): The average number of days it took for processing (setting the Disposition) of PBR notifications received by that office for the date range specified for the specific PBR Type identified.
As with all datagrids, information from the list may be viewed, exported to Excel or printed. By clicking  you may view and print. By clicking  a pop-up will appear in which you must click Open in order to view the information in Excel.

This datagrid supports all the common functionality for datagrids as described in Common User Interface Elements - Datagrids

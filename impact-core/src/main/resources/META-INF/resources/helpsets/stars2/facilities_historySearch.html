Facility History Search

Overview

The Facility History Search page, available from the Facilities tab, is a search page that gives you the ability to run a query to obtain HISTORICAL facility information for one or more facilities contained in the system. This search does not return CURRENT profile information for a facility. To search current profile information use the Facilities Search second-level menu under the Facilities tab. The Search Criteria and Result List for this search are described below.

This topic contains the following sections:

  • Facility History Search Criteria
  • Facility History Search Results
  • Related Help Topics

Facility History Search Criteria

You may search for HISTORICAL facility information on the basis of any or all of the criteria listed below.

  • Facility Name: Enter the whole name or a partial name and * or % in any position as a wildcard (example: An*, %Anh or *Anh*, etc.) to filter for an individual facility or group of facilities.

  • Facility Description: Enter the whole description or a partial description and * or % in any position as a wildcard (example: An*, %Anh or *Anh*, etc.) to filter for an individual facility or group of facilities.

Facility History Search Results

After you complete the criteria for the search you want, and click on  , the system searches all of the historical records for all the facilities stored in its database, selects the ones that meet ALL of the criteria you have specified, and returns summary information about those facilities in a datagrid. The datagrid below is the result of a search submitted with search criteria specified to select all facilities with a Facility Name that contains *university.

Facility History Search Results Datagrid

The summary data returned includes columns providing Facility ID, History ID, Facility Name, Facility Description, Start Date and End Date. The Start Date and End Date are in reference to the start and end period for the HISTORICAL Facility Detail.

Note that the data in the Facility ID and History ID columns in the datagrid are a hyperlink. If you click on a Facility ID in the datagrid, the system will take you to the CURRENT Facility Detail screen for the facility that the link you selected references. If you click on a History ID in the datagrid, the system will take you to the HISTORICAL Facility Profile screen for the facility that the link you selected references. You will know you are viewing a historical profile when you see the following at the top of the Facility Detail page:

** Historical version. Please note the facility inventory version start and end dates. **
